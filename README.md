# My Portfolio Website

Personal portfolio website built with Django.

<https://my-portfolio-wkh8.onrender.com/>

## About the Project

This project is a personal portfolio website developed using Python and Django.

The website is designed to present:

- Professional profile
- Work experience
- Technical skills
- Contact information
- Links to GitHub and LinkedIn

The project is being developed as part of my Python/Django learning journey and serves as a portfolio project.

## Technologies Used

- Python
- Django
- HTML5
- CSS3
- Bootstrap 5
- SQLite
- Git
- GitHub

## Features

### Profile Section

- Full Name
- Professional Title
- Location
- Phone Number
- Email Address
- Profile Photo

### Skills Section

- Add skills through Django Admin
- Display skills on homepage

### Experience Section

- Company Name
- Position
- Start Date
- End Date
- Current Position Status
- Description

### Administration

All content is managed through Django Admin Panel.
